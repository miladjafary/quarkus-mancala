Welcome To Mancala

For having fun with Mancala following steps is required.

  • Player 1 should start a new game by choosing New Game in the main menu
  • Player 1 should enter his name
  • The Game Id will be generated which can be shared with Player 2
  • Player 2 by choosing Join Game in the main menu, is able to enter the Game Id with his name
  • Player 1 and Player 2 will automatically redirect to the game board
That's it. Have fun with Mancala. Are you ready to start play Mancala?
